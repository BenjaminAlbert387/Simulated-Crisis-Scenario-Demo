## What this project is about
- This is a browser based visual novel demo that could be used to help heads and deputy heads identify and respond to staff members in a crisis.
- Created using RenPy
- All assets are copyright free
- Fully compatiable with laptop and mobile devices

## Requirements
- An internet browser and a Wi-Fi / mobile data connection

## Instructions
- Simply click on the link in the repository and the RenPy player should automatically load.
- Once you are ready, press Start!
